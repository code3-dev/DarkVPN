# ProxyCloud X (DarkVPN)

**ProxyCloud X** is a free, unlimited VPN service with no ads. It provides multiple locations worldwide with high speed and stability.

> ⚠️ Note: The source code is not included in this repository. Only the executable file is available.

## Features

- Free and unlimited VPN
- No advertisements
- Multiple server locations worldwide
- High speed and stable connections

## Compatibility

This app only supports **Android 7 to 13**.

## How to Use

1. Download the executable from this repository.
2. Run the program on your system.
3. Connect to your preferred server location.
4. Enjoy secure and private internet access.
